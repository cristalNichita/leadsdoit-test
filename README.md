<h1>Instruction:</h1>
<ul>
    <li>git clone</li>
    <li>composer install</li>
    <li>rename .env.example to .env</li>
    <li>php artisan migrate</li>
    <li>php artisan serve</li>
</ul>
<br>
<h1>API Links:</h1>
<p>Headers: x-token (check .env("XTOKEN"))</p>
<ul>
    <li>GET /api/weather?day={date: Y-m-d}</li>
</ul>
